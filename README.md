### Задача 1



### Текст задачи



#### Последовательно вводя числа, сформировать список из их последних цифр.



### Алгоритм решения

###### 

#### Запустить рекурсию "inputNum" в которой для завершения запроса чисел у пользователя надо ввести "ex". Дальше пользователь вводит число либо "ex". При вводе числа происходит преобразование строки в число, и берётся остаток от деления на 10, те последнее число и запускается новая рекурсия уже с добавлением полученной цифры в конец списка. При вводе непонятно чего потребует пользователя ввести заново то что он вводил.

# Задание 2



### Задача 1

### 

### Текст задачи



#### Найти минимальную цифру натурального числа



### Алгоритм решения



#### Запросить у пользователя число. Если число <10 то минимальное и есть то число что ввёл пользователь. Если больше то, берётся последняя цифра от введённого числа, берётся минимальная цифра в оставщейся части. После чего они сравниваются с помощью "min" и возвращает минимальную цифру.
